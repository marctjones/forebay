Created comprehensive interactive demo script for Forebay

**Demo Script Features:**
- Interactive walkthrough with colored output
- 8 complete real-world use cases
- Automatic environment setup and cleanup
- Environment variable configuration (WORKER_URL, CLI_PATH)
- Prerequisite checking (.NET, curl, jq)

**Use Cases Demonstrated:**
1. Health Check - Worker monitoring
2. Authentication - API key login/whoami
3. Basic Queue Ops - Push, pull, stats (task processing)
4. Job Queue - Background job processing (image pipeline)
5. Event Streaming - Real-time analytics (user events)
6. Multiple Queues - Priority management (high/normal/low)
7. Queue Lifecycle - Create, monitor, batch, delete
8. Direct API - REST API with curl examples

**Files Created:**
- `demo.sh` - Main executable demo script (~400 lines)
- `DEMO.md` - Complete documentation with troubleshooting

**Running the Demo:**
```bash
./demo.sh                          # Use production worker
WORKER_URL=http://localhost:8787 ./demo.sh  # Use local dev
```

**Demo automatically:**
- Checks prerequisites
- Builds CLI if needed
- Runs through all 8 use cases interactively
- Cleans up all demo data
- Shows summary and next steps

This provides a great onboarding experience for new users and showcases all implemented features!
