# Forebay Architecture & Issue Roadmap - Complete

## ✅ Clean Architecture Separation

The project is **already designed** with clean separation of concerns:

```
┌─────────────────────────────────────────────┐
│           Forebay.Core (Library)            │
│  ┌─────────────────────────────────────┐   │
│  │ - ForebayClient (API wrapper)       │   │
│  │ - Models (Request/Response)         │   │
│  │ - Exception handling                │   │
│  │ - HTTP communication                │   │
│  │                                      │   │
│  │ NO UI DEPENDENCIES                  │   │
│  │ Pure business logic                 │   │
│  └─────────────────────────────────────┘   │
└─────────────────────────────────────────────┘
              ▲          ▲          ▲
              │          │          │
    ┌─────────┴──┐  ┌────┴─────┐  ┌┴──────────┐
    │            │  │          │  │           │
┌───┴─────┐  ┌──┴──────┐  ┌───┴────────┐  ┌─┴──────────┐
│ Forebay │  │ Forebay │  │  Forebay   │  │  Forebay   │
│   CLI   │  │ Desktop │  │   Tasks    │  │   Notes    │
│         │  │         │  │  (shared)  │  │  (shared)  │
│ System. │  │Avalonia │  │            │  │            │
│ Command │  │   GUI   │  │ Terminal.  │  │ Terminal.  │
│  Line   │  │         │  │    Gui     │  │    Gui     │
└─────────┘  └─────────┘  └────────────┘  └────────────┘
   (v1.0)      (v1.1)        (v1.1)         (v1.1)
```

**Benefits:**
- ✅ CLI and GUI share **100% of business logic**
- ✅ Zero code duplication
- ✅ Easy to add new interfaces (mobile, web, etc.)
- ✅ Testable in isolation (no UI needed for tests)
- ✅ Consistent behavior across all interfaces

**Example:**
```csharp
// Forebay.Core - Pure library
var client = new ForebayClient("https://forebay.skpt.cl");
await client.LoginAsync(idToken);
var response = await client.PushAsync("work/tasks", payload);

// Forebay.Cli - CLI wrapper
Console.WriteLine($"Pushed: {response.Id}");

// Forebay.Desktop - GUI wrapper  
StatusLabel.Text = $"Pushed: {response.Id}";

// Same library, different UI!
```
